# Petek Kelime 🐝

Altıgen petek üzerinde harfleri sürükleyerek kelime bulma oyunu (prototip).

- 20 bölüm — her bölümde aynı konseptten 1-2 gizli hedef kelime
- Bonus kelimeler puan kazandırır ve ipucu çubuğunu doldurur
- Çubuk dolunca hedef kelimeden bir harf otomatik açılır
- Puanlama: 3→1, 4→3, 5→7, 6→13, 7→21, 8→31, 9→43, 10→57

## Çalıştırma

Tek dosya: `index.html`. Tarayıcıda açman yeterli, kurulum gerekmez.
